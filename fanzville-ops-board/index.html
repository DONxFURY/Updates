About this board
GitHub-ready

This is a static Ops Board built for GitHub Pages. Update the repo JSON files to publish changes:

  • /data/tasks.json — tasks and statuses
  • /data/updates.json — patch notes / announcements
  • /data/config.json — server name, links, and passphrase

Maintainer mode enables editing in-browser (stored in localStorage) and lets you export the JSON to commit back into the repo.

Tip: Change the maintainer passphrase in config.json and keep it private. This is not secure authentication — it’s just to prevent accidental edits.
Deploy to GitHub Pages
  1. Create a repo and push these files to main.
  2. Repo → Settings → Pages.
  3. Source: Deploy from a branch.
  4. Branch: main / Folder: / (root).

That’s it. Your Ops Board is live.

Customization
  • Colors and links: edit /data/config.json
  • Brand vibe: edit /assets/style.css variables
  • Add sections: edit /assets/app.js
